# Camera Calibration Checkerboard Pattern

Print this checkerboard on A4 paper for camera calibration.

## Instructions:
1. Print this pattern on A4 paper
2. Paste on rigid surface (cardboard/foam board) 
3. Use with camera_calibration.py tool
4. Capture 15-20 images at different angles and distances
## Pattern Specifications:
- **Size**: 9×6 internal corners (10×7 squares)
- **Square size**: 25mm (recommended)
- **Pattern type**: Checkerboard
